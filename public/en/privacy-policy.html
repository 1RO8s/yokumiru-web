プライバシーポリシー

最終更新日: 2026年6月24日

ヨクミル（以下「本アプリ」）は、利用者のプライバシーを尊重し、取得する情報を必要最小限に留めることを基本方針とします。

1. 取得する情報

本アプリは、機能提供のために以下の情報を利用する場合があります。

  • 利用者が登録したYouTube動画URL、動画ID、セクション名、再生履歴などのアプリ内データ
  • YouTube Data APIから取得する動画タイトル、サムネイル、チャンネル名、チャンネルアイコン、再生時間などの公開情報
  • Google連携を利用する場合、Googleアカウントの基本情報およびYouTube関連情報への読み取り権限
  • 有料プランを利用する場合、Apple App StoreおよびRevenueCatが提供する購入状態、購読状態、Entitlement情報
  • 問い合わせ時に利用者が入力したメールアドレス、問い合わせ内容、端末情報、添付ファイルなど

2. 情報の利用目的

取得した情報は、以下の目的で利用します。

  • 動画の保存、表示、再生、セクション管理、履歴表示など本アプリの機能提供
  • YouTube動画情報の取得および表示
  • 有料プランの購入状態確認、機能制限の解除、購入復元
  • 問い合わせ対応、不具合調査、利用者サポート
  • 本アプリの改善および品質向上

3. アプリ内データの保存

本アプリに登録された動画、セクション、再生履歴などのデータは、原則として利用者の端末内に保存されます。 本アプリは、独自の外部サーバーを用いてこれらのデータを収集・保管しません。

4. 外部サービスの利用

本アプリは、以下の外部サービスを利用する場合があります。

  • Google / YouTube API Services: YouTube動画情報の取得、Google連携
  • Apple App Store: アプリ内課金、サブスクリプション管理
  • RevenueCat: 購入状態およびサブスクリプション状態の管理
  • Google Forms: 問い合わせ受付

YouTube API Servicesの利用にあたっては、 YouTube利用規約、 Googleプライバシーポリシー、 Google利用規約 が適用されます。

5. 第三者提供

本アプリは、法令に基づく場合を除き、利用者の個人情報を本人の同意なく第三者へ提供しません。 ただし、外部サービスを利用する場合、それぞれのサービス提供者が定める規約およびプライバシーポリシーに従って情報が処理される場合があります。
